Learn Python

Variáveis

Variáveis armazenam valores.

nome = "Nochelli" idade = 25

Funções

Funções organizam código reutilizável.

def soma(a, b): return a + b

Loops

Repetem ações automaticamente.

for i in range(5): print(i)

Listas

Guardam múltiplos valores.

frutas = ["maçã", "banana"]

Tuplas

Tuplas armazenam múltiplos valores, mas não podem ser alteradas.

cores = ("azul", "verde", "vermelho") print(cores[0])

Dicionários

Guardam dados em chave e valor.

pessoa = { "nome": "Ana", "idade": 22 }

Classes e Objetos (POO)

Programação Orientada a Objetos (POO) organiza o código em objetos e classes.

class Pessoa: def __init__(self, nome): self.nome = nome def apresentar(self): print("Olá,", self.nome) p1 = Pessoa("Nochelli") p1.apresentar()

Herança

Uma classe pode herdar características de outra.

class Animal: def falar(self): print("Som do animal") class Cachorro(Animal): def falar(self): print("Au au") dog = Cachorro() dog.falar()

Métodos

Métodos são funções dentro de uma classe.

class Calculadora: def soma(self, a, b): return a + b calc = Calculadora() print(calc.soma(5, 3))

Encapsulamento

Encapsulamento protege dados internos da classe.

class Conta: def __init__(self): self.__saldo = 100 conta = Conta()

Busca Binária

def busca_binaria(lista, alvo): inicio = 0 fim = len(lista) - 1 while inicio <= fim: meio = (inicio + fim) // 2 if lista[meio] == alvo: return True elif lista[meio] < alvo: inicio = meio + 1 else: fim = meio - 1 return False numeros = [1, 3, 5, 7, 9, 11] print(busca_binaria(numeros, 7))

Função Recursiva

def fatorial(n): if n == 1: return 1 return n * fatorial(n - 1) print(fatorial(5))

Bubble Sort

numeros = [5, 2, 8, 1] for i in range(len(numeros)): for j in range(len(numeros)-1): if numeros[j] > numeros[j+1]: numeros[j], numeros[j+1] = numeros[j+1], numeros[j] print(numeros)
